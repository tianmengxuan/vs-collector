# VS振弦渗压计采集软件

基于**河北稳控科技 VS1XX/VS4XX 系列多通道振弦传感器无线采发仪**的云端数据采集系统。

---

## 功能特性

| 功能 | 说明 |
|------|------|
| 多协议解析 | 支持 HEX / STR1.0 / STR2.0 / STR3.0 数据帧 |
| TCP接收 | 设备通过4G/GPRS连接服务器，实时推送数据 |
| 振弦换算 | 自动将原始频率值换算为渗压(kPa)和水头(m) |
| 实时看板 | 按设备/通道展示最新水头值，颜色区分状态 |
| 历史曲线 | ECharts交互式曲线，支持缩放/平移 |
| 报警系统 | 可配置水头警告/报警阈值，自动记录 |
| 数据导出 | CSV格式导出，Excel可直接打开 |
| Docker部署 | 一键docker-compose部署到云服务器 |
| **桌面EXE版** | 双击运行，无需安装Python，适合Windows本地使用 |

## 方式一：Windows 桌面版 EXE（本地运行）

适合在**Windows电脑/工控机**上直接运行，双击即用。

### 打包步骤（需要一台 Windows 电脑）

```bat
# 1. 确保已安装 Python 3.9+（https://www.python.org）
# 2. 在项目目录双击运行打包脚本
build.bat
```

打包完成后，`dist\VS振弦渗压计采集软件.exe` 即为成品，可复制到任意目录运行。

### 使用说明

1. 双击 `VS振弦渗压计采集软件.exe`
2. 程序自动启动 TCP 采集服务（端口 8866）和 Web 界面（端口 5000）
3. 点击「**打开Web界面**」按钮，在浏览器中查看数据
4. 需要修改端口/报警阈值，点击「修改配置」

### EXE程序界面说明

| 区域 | 功能 |
|------|------|
| 左侧控制面板 | 服务状态指示、启动/停止/打开浏览器 |
| 运行日志区 | 实时显示设备连接和数据接收记录 |
| 修改配置 | 修改TCP端口、Web端口、报警阈值 |
| 打开数据目录 | 直接打开数据库和日志所在文件夹 |

> **注意**：EXE 版数据存储在程序同目录的 `data/` 文件夹，首次运行自动创建。

## 方式二：云服务器 Docker 部署（远程运行）

适合部署到**云服务器**，通过浏览器远程访问。

## 快速部署 (Docker - 推荐)

### 前提条件
- 云服务器已安装 Docker 和 docker-compose
- 开放端口: **8866** (TCP数据) 和 **5000** (Web界面)

### 步骤

```bash
# 1. 上传项目到服务器
scp -r vs-collector/ root@<服务器IP>:/opt/

# 2. 进入项目目录
cd /opt/vs-collector

# 3. 构建并启动
docker-compose up -d

# 4. 查看日志
docker-compose logs -f
```

访问 Web 界面: `http://<服务器IP>:5000`

## 设备端配置

在设备配置工具 **SETP** 中设置以下参数：

| 参数 | 配置值 |
|------|--------|
| 数据发送方式 | GPRS TCP |
| 监测数据包协议 | **字符串2.0** (推荐) 或 16进制协议 |
| TCP服务器地址 | `<您的服务器公网IP>` |
| TCP服务器端口 | `8866` |

> **重要**: 设备端"监测数据包协议"建议选 **字符串2.0 (STR2.0)**，可读性好，解析稳定。

## 传感器标定配置

编辑 `config.py` 中的 `CHANNEL_CONFIG`，填入每个渗压计的标定参数：

```python
CHANNEL_CONFIG = {
    1: {
        "name": "渗压计-1#",          # 传感器名称
        "desc": "坝体桩号0+200处",     # 安装位置
        "K": -0.001,                  # 标定系数 (来自传感器标定证书)
        "F0": 1245.0,                 # 基准频率 Hz (安装时初始频率)
        "B": 0.0,                     # 温度修正系数 (无修正填0)
        "T0": 25.0,                   # 基准温度 ℃
    },
    2: { ... },
}
```

### 渗压计换算公式

```
频率 F(Hz) = sqrt(原始值 × 100)
渗压 P(kPa) = K × (F² - F₀²) + B × (T - T₀)
水头 H(m)  = P / 9.8066
```

## 常见问题

**Q: 设备连上了但没数据？**  
A: 确认设备TCP协议设置正确，在设备上手动触发一次采发（按测试键），查看软件日志。

**Q: 水头数据显示异常？**  
A: 检查 `config.py` 中该通道的 `K`、`F0` 参数是否与传感器标定证书一致。

**Q: 如何查看设备UDID？**  
A: 设备启动日志中有15位IMEI码即为UDID，也可在Web界面"设备管理"页看到。
